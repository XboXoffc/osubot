# osubot
my osu bot with interesting functions

in tg: @xboxoffcbot

It has sqlite db, it working with osu api, writed on python, using asyncronous telebot for telegram integration
## Required modules
python 3.13,
telebot,
asyncio,
aiohttp,
sqlite3,
osu-tools-py,
validators
## Commands
general commands: https://telegra.ph/osubot-by-xbox-in-tgXboXoffcBot-01-05
